📊 Catalog Import / Export

Manage your price catalog via CSV spreadsheet. Export to edit in Excel or Google Sheets, then import back when done.

📥 Export Catalog CSV

Downloads your current catalog as a spreadsheet. Open in Excel or Google Sheets to edit prices, add items, change categories.

📤 Import Catalog CSV

Upload an updated catalog CSV. Replaces your current catalog. Use the exported file as your template.

CSV column format:   id, name, category, unit, material_cost, labor_hours   — first row must be the header. Leave id blank on new items and one will be assigned automatically.
📐 Templates Import / Export

Manage job templates via CSV. Each row is one line item — template header info repeats per row.

📥 Export Templates CSV

Downloads all templates as a spreadsheet. Edit existing templates or add new ones, then import back.

📤 Import Templates CSV

Upload an updated templates CSV. Replaces your current templates.

CSV column format:   template_id, template_name, icon, category, environment, margin, item_desc, item_category, qty, unit, mat_cost, labor_hrs   — one row per line item. Repeat template_id/name/icon/category/environment/margin on every row that belongs to the same template.
💾 Full System Backup

Export or restore your entire system — quotes, jobs, customers, catalog, templates, settings — as a single JSON file.
